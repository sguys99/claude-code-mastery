# Task 007: 반응형 디자인 적용

## 목표

모바일 및 태블릿 환경에서 최적의 사용자 경험을 제공하기 위한 반응형 디자인 개선

## 작업 내용

### 1. 견적서 페이지 반응형 개선

- [x] 견적서 헤더 (제목/날짜) 모바일 레이아웃 개선
- [x] 견적 항목 테이블 모바일 최적화 (카드 형태로 변환)
- [x] 상단 버튼 영역 모바일 최적화

### 2. 전체 페이지 반응형 검증

- [x] 홈 페이지 모바일 뷰 검증
- [x] 404 에러 페이지 모바일 뷰 검증
- [x] 500 에러 페이지 모바일 뷰 검증 (구조 동일)

## 구현 상세

### 견적서 헤더 개선

- 모바일: 제목과 날짜를 세로 배치
- 데스크톱: 기존 가로 배치 유지

### 테이블 반응형 전략

- 테이블에 최소 너비 설정 (min-w-[600px])
- 가로 스크롤 힌트 추가
- 또는 모바일에서 카드 형태로 변환

## 검증 기준

- [x] 모바일 (375px) 뷰에서 모든 콘텐츠 정상 표시
- [x] 태블릿 (768px) 뷰에서 최적화된 레이아웃
- [x] 데스크톱 (1024px+) 뷰에서 기존 레이아웃 유지
- [x] Playwright MCP로 각 뷰포트 테스트 완료

## 기술 스택

- TailwindCSS v4 반응형 유틸리티 (sm:, md:, lg:)
- 모바일 퍼스트 접근법
